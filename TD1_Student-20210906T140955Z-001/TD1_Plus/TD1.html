Les titres et sous-titres

Ces balises titre h1, h2, ..., h6 indiquent le niveau du titre : h2 suit h1, h3 suit h2 etc. Une grosse erreur consisterait à croire que vous allez utiliser h1, h2 ou h6 en fonction de la taille du texte que vous voulez obtenir. L'habillage de vos balises pourra dans un second temps être réglé par l'utilisation d’une feuille de style CSS (bientôt dans les TD prochains).

Les médias en HTML

Images

On peut ajouter les images avec la balise de type en-ligne < img> qui permet de mettre une image dans un texte, c'est-à-dire qu'elle s'affiche au fil du texte, Les balises de type en ligne s'affichent les unes à la suite des autres. Les balises de type bloc s'affiche les unes sous les autres

On peut aussi ajouter les images avec la balise de type bloc < figure> et où on peut aussi mettre un titre de l'image

Vidéos

Pour ajouter les vidéos, on utilise la balise < video> . Seul les formats MP4, WebM et Ogg sont supportés en HTML

Documentation W3school ICI

Audios

Pour ajouter les audios, on utilise la balise < audio>. Seul les formats MP3, WAV et OGG sont supportés en HTML

Les tableaux

Voici les balises à retenir pour créer un tableau en HTML:

  • Balises élémentaires
    • table : crée un tableau
    • caption : légende du tableau (optionnel)
    • tr : ligne du tableau (tr signifie "table row", soit en français une ligne de tableau)
    • th : cellules d’en-tête du tableau (th signifie "table head", soit en français un en-tête de tableau)
    • td : cellules du tableau (td signifie "table data", soit en français une donnée du tableau)
  • Balises complémentaires
    • thead : groupe de cellules d'en-tête
    • tbody : groupe de cellules de corps de tableau
    • tfoot : groupe de cellules de pied de tableau
  • Propriétés permettant de coller des cellules
    • colspan : groupe des colonnes, par exemple : < td colspan="2"> (cette cellule et celle située juste à côté à droite sur la même ligne sont regroupées en une cellule)
    • rowspan : groupe des lignes, par exemple : < td rowspan="2"> (cette cellule et celle située juste en dessous sont regroupées en une cellule)
